# 自检清单

## 入口自检 — 这是复盘吗？
- [ ] 是基于自己过去真实做过的事吗？
- [ ] 是为了未来做类似事情更好吗？
- [ ] 目的是让自己/团队变强吗？

## 深度自检 — 我挖到哪一层了？
- [ ] 我说得出「这次感觉怎么样」→ L1
- [ ] 我有事实支撑（有数据更好，实在没有有事实也行）→ L2
- [ ] 我说得出造成结果的原因 → L3
- [ ] 我总结出了可复用的规律/模型/清单 → L4
- [ ] 我发现了团队/自己的核心能力短板 → L5

## 跑偏自检 — 我跑偏了吗？
- [ ] 我是不是做成功了就直接庆祝/归因运气了，没复盘？→ 成功逃避式
- [ ] 我是不是全在说别人的问题，没说自己？→ 背锅式
- [ ] 我是不是只有数字没有分析和结论？→ 数字打卡式
- [ ] 我是不是只有感受没有事实？→ 主观感受式
- [ ] 我是不是全在优化细节，没跳出来看策略？→ 细节陷溺式
- [ ] 我是不是不好意思提真问题，结论都很模糊？→ 和气模糊式
- [ ] 我是不是只说好话，回避了失败和不足？→ 自我感动式
- [ ] 我是不是只挑毛病，完全没说哪里做得好？→ 只挑毛病式
- [ ] 我上次复盘写的行动，后来做了吗？→ 尘封式
