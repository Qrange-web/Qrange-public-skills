# 评分规则

## 各层评分门槛

| 层级 | 通过标准 | 未通过时 |
|------|---------|---------|
| L1 | 能说出明确的感受方向（正向/负向/混合） | 引导：「哪怕只有一个词，你觉得这次是顺还是卡？」 |
| L2 | 能列出至少 2 个具体事实（时间/地点/数字/动作/原话，有数据更好） | 标记「缺事实」，继续但不标注 L2 已通过 |
| L3 | 根因链条至少深挖两层；成功复盘需区分「刻意做到」vs「可能运气」 | 标记「卡在 L3」，标注障碍类型 |
| L4 | 产出至少一条可复用的规律/模型/框架/清单 | 标记「待深挖：缺积累/缺案例」 |
| L5 | 能说出一个能力短板，且能解释补齐后其他事也会改善 | 不强求，未通过不标记为问题 |

## 整体评分

| 等级 | 条件 |
|------|------|
| ✅ 合格 | L1-L3 全部通过 |
| 🌟 优秀 | L1-L3 全部通过 + L4 通过 |
| 🏆 卓越 | L1-L5 全部通过 |
| ⚠️ 待深挖 | L3 未通过 |
| ❌ 无效 | L1 未通过（极少见） |

## Wiki 标记触发规则

- **待深挖**：L3 未通过，或原则自检亮起信号（缺事实/缺外部视角/没明确预期/视角单一）
- **已沉淀**：L4 通过（硬门槛——必须有至少一条 L4 级产出）
