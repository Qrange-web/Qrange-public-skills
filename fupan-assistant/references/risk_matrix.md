# 风险分级

## 风险四级

| 等级 | 场景 | Skill 行为 |
|------|------|-----------|
| 🟢 低风险 | 正常复盘，用户情绪平稳，L1-L3 顺利推进 | 标准追问流程 |
| 🟡 中风险 | 用户归因过于外部化（甩锅倾向）；过于自我批评（情绪崩塌倾向）；涉及第三方评价 | 切换追问策略，不强行推进。复盘卡标注「仅基于个人视角」 |
| 🔴 高风险 | 涉及法律/合规问题；用户准备把复盘卡当作归责证据 | 拒绝输出归因结论。不输出复盘卡。标注：「本复盘仅用于个人学习，不构成对任何第三方的评价或归责。如有合规问题，请咨询专业人士。」 |
| ⛔ 不触发 | 不属于复盘的请求 | 礼貌告知边界，给替代建议 |

## 方法论特有风险

| 风险 | 具体表现 | 处理方式 |
|------|---------|---------|
| 用户被追问后情绪崩塌 | 失败事件复盘触及痛点，用户陷入自我攻击 | 停止追问根因，切换为「面向未来表达」 |
| 复盘变成甩锅 | 用户把所有原因归于外部，不反求诸己 | 追问「你自己能控制的部分里，哪个决策可以做得更好」，继续回避则标记「缺外部视角」 |
| Skill 被当作背锅工具 | 用户产出一份「结论是别人有问题」的复盘卡 | 复盘卡标注「本复盘仅基于你的个人视角」 |
| 浅层复盘被标记为已沉淀 | 用户只挖到 L2 就满意了 | 已沉淀硬门槛：必须有 L4 级产出 |
| 复盘成瘾但不执行 | 每次聊很深但从不落地 | 结尾追问上次复盘的行动执行情况，连续两次未执行则提醒 |
