# 证据等级规则

未来补充案例或验证主张时遵守以下规则：

1. **优先使用**：官方来源（企业官网、研究机构报告）、证券交易所公开数据、同行评议的学术论文
2. **可以使用但需标注**：行业书籍/MOOC 课程中引用的案例（标注「企业实践，未经独立验证」）、课程平台内部案例（标注「课程案例，来自主讲人个人经验」）
3. **禁止引用**：自媒体文章、营销软文、无法核实来源的「某公司/某人」案例
4. **每个案例必须标注**：来源类型（课程案例/外部案例/用户案例）、可信度等级（⭐⭐⭐/⭐⭐/⭐）、是否经过独立验证
5. **用户案例脱敏规则**：从实际对话中积累的案例，必须去除所有可识别信息（姓名、公司名、具体数字范围），仅保留事件类型和方法论价值
