# tizhinei-writing · 体制内文风写作 Skill

> 一个可以装进任何 AI Agent 的文风转换技能：把文本改写或起草成符合中国体制内（党政机关、央国企）官方公文风格的文字——权威、严谨、克制、有指导性。

**它不是"换几个高级词"，而是一套完整的公文语言规则系统。**

---

## 解决什么问题

AI 默认写出来的"正式中文"常常四不像：

- **学生腔混公文体**：有"赋能""抓手"，也有"我觉得""特别棒"
- **主语乱飞**：满篇"我们公司""我们将"，一看就不是机关行文
- **细节过满**：写了大量"怎么做"的操作细节，失了"指方向、定原则"的高度
- **语气不笃定**："可能会进一步提升"，公文里没有"可能"

这个 Skill 把体制内文风的隐性规则显式化：省略主语、权威动词库、四字格并列、战略抽象、必然性语气——外加一份输出自检清单，写完逐项过。

---

## 快速开始

### 安装

这是一个标准的 Agent Skill 包（`SKILL.md` + `references/`），平台无关：

- **支持技能包格式的 Agent 运行时**（Claude Code、OpenClaw、Hermes、WorkBody，或任何兼容 SKILL.md 规范的个性化 Agent）：把整个 `tizhinei-writing` 文件夹放进对应的 skills 目录即可
- **不支持技能包的 Agent**：把 `SKILL.md` 全文作为系统提示词注入，`references/examples.md` 作为上下文按需提供给模型

### 触发

让用户直接说（任意一句）：

```
帮我写成汇报材料
改成机关文风
润色成公文
正式一点，像政府文件那样
```

或直接粘贴一段文字，说"改成体制内风格"。支持两种模式：**改写**（已有文本→公文风格）和**起草**（给主题要点→直接撰写）。

## 工作原理

### 五条文风规则

| 规则 | 一句话说明 |
|---|---|
| **省略主语** | 不用"我们""公司将"，默认主语为报告主体，动词开头推进 |
| **权威动词** | "做"→落实/推进/抓好；"建"→构建/夯实/筑牢；"改"→优化/深化/完善 |
| **并列结构** | 四字格 + 排比；"一是""二是""三是"各分项总起句字数一致、结构对仗 |
| **战略抽象** | 聚焦"做什么"（What），不提"怎么做"（How），为执行层留出空间 |
| **必然性语气** | 避用"可能""或许"，强调战略必要性和因果必然性 |

### 四类常用句式库

战略引导类（"坚持……战略导向，聚焦……核心任务"）、工作部署类（"着力推动……，切实提升……"）、成果总结类（"初步构建起……体系，形成……格局"）、问题分析类（"对照……要求，当前……仍面临多重制约"）。

### 输出自检

写完不过夜——六条自检逐项过：第一人称主语？不确定性词？动词够不够权威？分项总起句对仗？操作细节是否越位？语气是否克制？

`references/examples.md` 提供 4 段脱敏真实公文 + 逐条标注，展示每条规则落在纸面上长什么样。

## 边界（它不做什么）

- **不替你编造事实**：改写模式保留原文所有关键事实和数据，不擅自增删；起草模式信息不足时会先追问
- **不保证"像某个具体单位"**：规则提炼自通用公文惯例，不模拟特定机关的行文癖好
- **不做内容审查判断**：文风是文风，立场是立场——它管文字像不像公文，不管内容该不该发
