# 平衡计分卡 — 课程主线

## 1. 课程一句话定位

**平衡计分卡不是一套业绩衡量指标，而是一个战略管理系统**。 它用四个层面（财务、客户、内部流程、学习与成长）的因果关系链，把企业战略从会议室里的模糊口号，变成每个员工每天都能操作的日常行动。

## 2. 课程要解决的核心问题

> 战略制定得很好，为什么执行总是失败？

传统管理系统围绕**财务会计框架**构建，而财务会计只能衡量有形资产和过去的结果。信息时代的价值创造主要靠无形资产（客户关系、创新能力、员工技能、信息系统）——这些东西，财务会计完全看不见。结果是：

| 障碍 | 症状 |
|------|------|
| 愿景无法转化为行动 | 高层知道战略，中层和一线只盯着短期财务指标 |
| 战略未衔接个人目标 | 奖金仍跟短期利润走，没人管长期能力建设 |
| 战略未衔接资源分配 | 战略计划锁在抽屉里，预算另搞一套 |
| 只有战术反馈没有战略反馈 | 每月开会只讨论预算差异，不讨论战略本身对不对 |

**根因**：不是执行的人不行，是**管理系统本身没有为战略执行设计**。

## 3. 核心方法

### 3.1 概述

平衡计分卡通过四个层面的因果关系链，把战略变成可操作的指标体系：

```
学习与成长（基础：员工能力、信息系统、组织氛围）
    ↓
内部业务流程（你能做什么：创新→经营→售后服务）
    ↓
客户（客户怎么看你：满意度、保持率、市场份额、获利率）
    ↓
财务（股东得到什么：收入增长、成本降低、资产利用）
```

每个下层指标是上层指标的**驱动因素**。不是并列关系，是**因果关系链**。

### 3.2 四个层面详解

#### 财务层面——定方向

不是所有业务单位都用同一个 ROI。企业生命周期决定财务目标：

| 生命周期 | 战略重点 | 核心指标 |
|---------|---------|---------|
| **成长期** | 占领市场 | 收入增长率、新产品收入占比、目标市场销售增长率 |
| **保持期** | 获利最大化 | 资本报酬率、EVA、毛利率、客户/产品线利润率 |
| **成熟期** | 现金回收 | 经营现金流量、现金周转期、回收期 |

三个财务主题贯穿所有阶段，但具体指标不同：

| 主题 | 成长期 | 保持期 | 成熟期 |
|------|--------|--------|--------|
| 收入增长 | 细分市场增长率、新产品收入占比 | 目标客户占有率、交叉销售 | 客户/产品线利润率 |
| 降低成本 | 人均收入 | 相对于竞争者的成本、成本降低率 | 单位成本 |
| 资产利用 | 投资占销售额%、研发占比 | 营运资金比率、资产利用率 | 回收期、生产能力利用率 |

**关键指标：现金周转期** = 库存天数 + 应收账款天数 - 应付账款天数。这个指标衡量的是：钱从付给供应商到从客户收回，到底卡了多久。

#### 客户层面——定战场

**第一步：选客户**。 战略的本质是选择有所为、有所不为。不是所有客户都要服务。

**核心指标链**（五个指标构成因果关系）：

```
客户满意度 → 客户保持率 + 客户获得率 → 市场份额
                                        ↓
                                  客户获利率（确保份额是有利可图的）
```

**客户价值主张三要素**：

| 维度 | 内容 | 决定的是 |
|------|------|---------|
| 产品/服务特征 | 功能、价格、质量 | 客户得到什么 |
| 客户关系 | 交货体验、响应时间、个人关系 | 客户怎么得到 |
| 形象和声誉 | 品牌、信任 | 客户为什么选你 |

**客户获利率矩阵**：把客户按"是否目标细分市场"和"是否获利"分成四象限——目标且获利的保留，目标但非获利的转变，非目标但获利的监控，非目标且非获利的消除。

#### 内部业务流程层面——定能力

**最大突破**：不只改善现有流程，更要确认**必须擅长但尚未建立的新流程**。

三个流程构成价值链：

```
创新流程（长波）→ 经营流程（短波）→ 售后服务流程
```

- **创新流程**：确认客户需求 → 开发新产品/服务。关键指标：**BET 收支平衡时间**（从产品开发到产生足够利润收回研发成本所需时间）
- **经营流程**：订单到交付。关键指标：**MCE 制造周期效率** = 加工时间 / 产出时间（理想值趋近于 1，即所有时间都是增值的）
- **售后服务流程**：保修、维修、收款。关键指标：一次成功率（不返工即可满足客户要求的比例）

#### 学习与成长层面——定根基

三个来源：

| 范畴 | 核心内容 | 关键指标 |
|------|---------|---------|
| 员工能力 | 技术再造、技能提升 | **战略工作胜任率**（符合战略工作资格的员工/预期需要的人数） |
| 信息系统能力 | 信息获取、数据支持 | **战略信息覆盖率**（可用信息/预期需求） |
| 激励、授权与协作 | 组织氛围、目标一致 | 员工建议采纳率、个人目标一致性比率 |

**员工核心指标链**：员工满意度 → 员工保持率 → 员工生产率

**关键工具：半程量尺**。衡量流程缺陷减少 50% 所需的时间。假设持续改进以固定速率减少缺陷，那么半程量尺告诉你：按当前速度，能不能在预定时间内达成目标。

### 3.3 因果关系链是灵魂

**三个原则**：

1. **因果关系**：每个指标都是因果链的一环，最终指向财务目标。战略就是一套"如果……那么……"的假设
2. **成果 + 动因混合**：只有结果指标（滞后指标），不知道"怎么做到的"；只有动因指标（领先指标），不知道"做到了又怎样"
3. **必须与财务挂钩**：所有改进的最终检验标准是它是否转化为财务结果。质量改进如果不能变为利润，就不是真正的成功

**一个完整的因果链示例**：

```
员工培训投入 ↑ → 战略工作胜任率 ↑ → 内部流程质量 ↑ → 按时交货率 ↑
→ 客户满意度 ↑ → 客户保持率 ↑ → 市场份额 ↑ → 资本报酬率 ↑
```

### 3.4 战略指标 vs 诊断指标

| 类型 | 数量 | 作用 | 管理方式 |
|------|------|------|---------|
| **战略指标** | 20-25 个 | 定义战略、创造竞争优势 | 高层管理者反复研讨、辩论 |
| **诊断指标** | 数百个 | 监控正常运转，异常时报警 | 例外管理 |

**比喻**：体温和血压是诊断指标（重要但非竞争优势），专业能力是战略指标（决定职业成功）。平衡计分卡只放那 20-25 个战略指标，其余几百个诊断指标仍然存在，但不进计分卡。

### 3.5 不同组织形态下的平衡计分卡

平衡计分卡不仅适用于单一战略业务单位（SBU），书中第八章专门讨论了四种组织形态的适配：

#### 总公司级计分卡

拥有多个 SBU 的大公司需要总公司级计分卡来回答一个问题：**为什么要把这些 SBU 放在一个公司里，而不是让它们各自独立经营**？

| 协同类型 | 内容 | 示例 |
|---------|------|------|
| **共享客户** | 多个 SBU 服务同一客户群 | 强生公司全球 150+ 经营公司，共享医疗保健客户 |
| **共享技术** | 核心能力跨 SBU 复用 | 本田的发动机能力横跨摩托、汽车、割草机；佳能的微电子/缩影能力横跨相机、复印机、打印机 |
| **共享职能** | 采购、财务、IT 集中以获得规模经济 | 凯尼恩商店的房地产部集中租赁谈判 |
| **总公司主题** | 所有 SBU 共同遵守的价值观和原则 | 杜邦的安全保障、3M 的创新主张 |

**总公司级计分卡的两个要素**：
1. **总公司主题**：价值、信仰，所有 SBU 必须共同遵守
2. **总公司作用**：为创造 SBU 之间的协同效果而采取的行动（交叉销售、共享技术、共享服务）

#### 合资企业与战略联盟

平衡计分卡帮助界定合资方的**共同目标**。书中案例：石油技术公司（数家油田服务公司组成的合资企业），核心指标是"每桶油的生命周期成本"——一个绝佳的客户结果指标，它清楚显示了合资企业是否比各自独立经营创造了更多价值。

#### 共享职能部门（后勤/行政部门）

> "行政人员提供了哪些竞争优势？如果两者都没提供，我们就应该考虑把这个职能部门外包出去。"

共享职能部门（IT、HR、采购等）必须回答：**我们提供的是低成本还是差异化服务**？如果两者都不是，应该外包。书中案例：某电信公司信息支援部（IT 部门），通过构建以内部客户为中心的平衡计分卡，从"指定供应商"转型为"以客户为中心的竞争者"，区分了第一级客户（要创新、要长期合作）和第二级客户（要低成本、高可靠）。

#### 政府机构与非营利组织

**关键区别**：财务是约束，不是目标。成功取决于能否有效满足利益相关者需求。

书中案例：
- 美国联邦政府采购主管协会（PMAT）：在传统四层面基础上增加了第五个层面——**员工授权**
- 加州桑尼维尔市：连续 20 年发布包含目标、社区指标、业绩指标的年度报告，主管超额完成可获 10% 奖金
- 北卡罗莱纳州夏洛特市：1995 年发布年终目标计分卡，涵盖社区安全、经济发展、交通等五个重点区域
- 马萨诸塞州残疾人奥林匹克委员会：财务层面焦点是捐款人期望，客户层面焦点是运动员，内部流程关注组织管理和公关，学习与成长关注志愿者和教练培训

## 4. 操作流程

### 建立平衡计分卡（十步法）

```
Step 1-2: 确定衡量结构
  ├─ 选择适当的战略业务单位（SBU，拥有完整价值链和独立战略）
  └─ 确定 SBU 与总公司的联系（财务目标、公司主题、跨 SBU 协同机会）

Step 3-5: 建立对战略目标的共识
  ├─ 第一轮座谈：与每位高层管理者 90 分钟，收集战略意见
  ├─ 总结会议：综合座谈结果，草拟试验性目标和指标
  └─ 第一阶段执行研讨会：就使命、战略、目标达成共识，选出最重要的 3-4 个目标

Step 6-7: 选择和设计指标
  ├─ 子团队会议：四个层面分别由子团队负责，确定指标、确认信息源、建立因果关系
  └─ 第二阶段执行研讨会：子团队汇报，讨论指标，构思挑战性目标值

Step 8-10: 制定实施计划
  ├─ 开发实施计划：确定目标值，建立信息系统，设计沟通方案
  ├─ 第三阶段执行研讨会：最终共识，核准目标值，确定初步行动方案
  └─ 完成实施计划：60 天内开始使用，用"目前最好的信息"起步
```

总时长约 **16 周**。

### 用平衡计分卡管理战略（四大管理流程）

平衡计分卡的价值不只是"建立"的十步，更是"使用"的四大管理流程。这四步共同构成了一个完整的战略管理系统，填补了传统管理系统中"战略执行"的空白。

#### 流程一：沟通与联系——让所有人知道战略

> 战略实施的第一个障碍：愿景无法付诸行动。调查显示，只有 40% 的中层管理者了解公司愿景，不到 30% 的奖金与长期战略挂钩。

**三种协调机制**：

1. **沟通与教育计划**（内部营销）
   - 目标：让员工了解战略，促使他们主动采取行动实现战略目标
   - 工具：战略宣传册、员工大会、简讯、录像、电子公告栏、群组软件
   - 先锋石油公司：每月简讯开辟平衡计分卡专栏，先教育后反馈，每期报道一个层面的最新成绩，并讲述部门/个人为取得这些成绩做贡献的故事
   - 凯尼恩商店：为不同对象（总公司、SBU 领导、店长、配送中心、供应商）定制不同的沟通渠道和频率

2. **目标制定计划**（层层下达）
   - 总公司 → SBU → 部门 → 团队 → 个人
   - **个人口袋计分卡**：三层信息——总公司目标（左）、业务单位目标（中）、个人目标与行动（右），可放进衬衫口袋随身携带
   - 关键：平衡计分卡的因果关系框架，为层层分解非财务指标提供了逻辑依据——每层都知道自己的指标驱动了上一层什么

3. **与奖励制度挂钩**（谨慎但必要）
   - 先锋石油公司：60% 奖金与财务指标挂钩（5 个加权指标），40% 与客户/内部/学习与成长挂钩
   - **门槛条件法**：所有战略指标达到最低门槛才发奖金，防止"几个指标超额完成、其他一塌糊涂仍拿满奖金"
   - **主观判断法**：像跳水比赛一样，高层管理者根据目标值难度和完成情况主观评分
   - 风险：指标选错、数据不可靠、追求指标的手段违背初衷——所以建议积累经验后再挂钩，但短期必须至少**停止纯粹按短期财务结果发奖金**，否则言行不一

#### 流程二：计划与目标值——让资源服务于战略

**四个步骤**：

1. **制定挑战性目标值**（3-5 年）
   - 目标值应代表业务单位的业绩突破——如果公司是上市公司，目标值的实现应导致股价成倍上涨
   - 关键：不是孤立地给每个指标设目标，而是通过**因果关系链**让管理者看到：如果完成了客户/内部/学习层面的目标，财务业绩自然会突破。这解决了"财务目标激进但营运目标保守"的脱节

2. **确定及合理化战略行动方案**
   - 三种行动方案类型：

   | 类型 | 触发条件 | 方法 |
   |------|---------|------|
   | **"衡量指标缺失"方案** | 计分卡指标缺乏数据 | 建立新的管理流程来收集数据。指标的缺失暴露了管理流程的缺失 |
   | **持续改进方案** | 渐进改善可实现目标 | 用半程量尺跟踪改进速率 |
   | **战略行动方案（再造/转型）** | 持续改进不足以实现突破 | 根本性重新设计流程。与传统的以削减成本为目标的再造不同，BSC 的再造目标可能是缩短周期、提升能力等非财务指标 |

   - 案例：大都会银行建立计分卡后，从 70 多个行动方案中砍掉了与战略无关的方案，包括一个针对高收入人士的营销计划和一个加强低水平销售技术的方案，取而代之的是与战略一致的技术再造计划

3. **确定跨业务单位行动方案**：利用协同效应，如凯尼恩商店的房地产部集中租赁谈判

4. **战略预算与经营预算合一**：五年计划的第一年 = 年度预算。短期里程碑（月度/季度非财务目标）也纳入预算编制，而不仅仅是财务数字

#### 流程三：战略反馈与学习——让战略能自我修正

> 这是全书最具创新性的部分。大多数企业的定期会议只讨论经营问题（实际 vs 预算差异），几乎不讨论战略本身对不对。

**单循环 vs 双循环学习**：

| | 单循环学习 | 双循环学习 |
|--|-----------|-----------|
| **做什么** | 发现执行偏差 → 纠正 → 回到预定轨道 | 质疑战略假设本身 → 反思经营理论 → 修正战略 |
| **前提** | 战略是正确的 | 战略可能错误 |
| **触发信号** | 业绩驱动因素未达标 | 业绩驱动因素达标了，但成果指标没改善 |
| **对应会议** | 月度经营研讨会 | 季度战略研讨会 |

**五种战略学习机制**：

| 机制 | 内容 | 示例 |
|------|------|------|
| **相关分析** | 检验计分卡指标之间的统计相关性 | 回声工程公司发现：员工士气↑→客户满意度↑→应收账款天数↓（从 60 天降到 30 天）→资本报酬率↑ |
| **管理游戏与情景分析** | 基于计分卡模型模拟不同战略情景 | 管理层用第一年数据构建模型，评价战略缺陷，模拟改进方案 |
| **轶事报道** | 讲述数字背后的故事，获得非正式反馈 | 大都会银行每月简讯报道某个销售人员如何成功建立新客户关系 |
| **行动方案研讨** | 持续评价战略行动方案对成果指标的影响 | 判断一组行动方案（媒体广告+信用卡推广+使用率提升）对成果指标（新账户数+活跃账户率）的实际贡献 |
| **同行研讨** | 其他业务单位管理者独立评价计分卡 | 某电子元件公司每半年由 5-6 位其他分公司经理组成小组，回顾战略、指标、行动方案，并随机访问员工 |

**经营研讨会 vs 战略研讨会**：

| | 经营研讨会 | 战略研讨会 |
|--|-----------|-----------|
| 频率 | 每月 | 每季度 |
| 时间分配 | 50% 研讨业绩，40% 讨论影响，10% 战略 | 60% 调整战略，30% 讨论影响，10% 研讨业绩 |
| 核心问题 | "我们有没有按计划执行？" | "我们的战略还对不对？" |

**关键**：两个会议必须分开。如果混在一起，战术问题永远挤压战略问题。

#### 流程四：一体化管理——让系统持续运转

> 30 个月完成从建立计分卡到全面融入管理系统的转变。以国家保险公司为例：第 1-3 月诠释愿景 → 第 4-6 月开发 SBU 计分卡 → 第 7-9 月研讨+撤销非战略性投资 → 第 10-12 月推出变革方案+全面推广 → 第 13-24 月个人目标挂钩+月度/季度研讨 → 第 25-30 月修订长期计划+全员奖金挂钩。

**三个过渡角色**：

| 角色 | 职责 | 谁来做 |
|------|------|--------|
| **设计师** | 建立最初的平衡计分卡，将计分卡引入管理系统 | 通常是战略计划/质量管理/财务 VP，可配外部顾问 |
| **改革者** | 将计分卡融入日常管理流程，确保新管理系统运行 | 应与 CEO 直接汇报，在 2-3 年内指挥新系统进展 |
| **沟通者** | 争取所有员工对计分卡的认知、接受和支持 | 专人负责，沟通部门大力支持，当作内部营销来做 |

**谁来持续管理战略系统**？ CEO 是最终的流程主人，但日常运行需要专人。传统职位中 CFO（太偏财务控制）和战略规划部经理（太偏年度事件）各有限制，作者预判这会成为一个新的重要职位。

## 5. 判断标准

| 检验项 | 通过标准 | 不通过时 |
|--------|---------|---------|
| 财务指标是否匹配生命周期 | 成长期看收入增长，保持期看资本报酬率，成熟期看现金流 | 用一个 ROI 考核所有业务单位——该投资的没投，该收割的没收割 |
| 客户指标是否区分了目标市场 | 计分卡上明确写了"为哪些客户创造什么价值" | 客户满意度、市场份额等指标通用但没有细分——实际是在为所有人服务，等于没服务任何人 |
| 内部流程是否包含创新流程 | 计分卡上既有经营流程指标，也有新产品开发、市场调研等创新流程指标 | 只衡量现有流程的效率和质量——拿了满分也只是把昨天的事做得更好，明天的竞争力没建 |
| 学习与成长层面是否有具体指标 | 有战略工作胜任率、战略信息覆盖率等可量化的指标 | 只有"加强培训""提高员工满意度"等口号——无法判断投入够不够、方向对不对 |
| 因果关系链是否完整 | 每个指标都能追溯到它驱动什么、被什么驱动，最终指向财务目标 | 四个层面的指标互相独立，看不出任何先后因果——只是一堆指标的集合，不是战略的翻译 |
| 成果指标和动因指标是否混合 | 既有市场份额、利润率等滞后指标，也有面对客户时间、交叉销售率等领先指标 | 全是滞后指标→不知道结果怎么来的；全是领先指标→不知道改进有没有转化为财务结果 |
| 战略研讨会是否独立于经营研讨会 | 每季度一次战略研讨会，专门讨论战略有效性和调整，不与月度经营分析混在一起 | 每次开会 90% 时间讨论预算差异，战略议题被挤压到"下次再说" |
| 奖金是否与平衡计分卡挂钩 | 奖金至少部分与计分卡的非财务指标挂钩，且设有门槛条件（所有指标达标才发奖金） | 奖金仍 100% 跟短期财务指标走——员工嘴上说战略重要，身体很诚实地只追短期利润 |

## 6. 适用场景与边界

### 适用场景

- **需要战略转型的企业**：现有战略不清晰，各部门各自为政，需要统一方向
- **多元化企业中的战略业务单位**：拥有完整价值链（创新→经营→营销→服务）、独立战略、独立客户和渠道的 SBU 最合适
- **无形资产占比高的企业**：服务业、科技企业、金融业——这些企业的价值创造主要靠人、客户关系和信息系统，而非机器厂房
- **战略制定清晰但执行不力的企业**：高层有共识，但中层和一线不知道战略是什么、自己该做什么
- **从命令控制型管理转向学习型组织的企业**：需要双循环学习能力，能质疑和调整战略本身

### 不适用场景

- **纯成本中心或执行部门**：没有独立战略，只是执行上级指令的部门不需要计分卡，关键业绩指标（KPI）就够了
- **战略本身尚未明确的组织**：平衡计分卡是战略实施的机制，不是战略制定的机制。如果连为谁服务、差异化在哪都不清楚，先做战略规划
- **初创期微型企业**：规模太小，战略沟通靠口头就能完成，不需要一个 20-25 个指标的系统
- **短期现金流极度紧张的企业**：当生存是唯一问题时，平衡计分卡的长周期建设来不及救命
- **试图复制其他公司计分卡的企业**：每个计分卡必须从自身战略出发定制。复制最佳公司的指标，等于照着别人的药方给自己抓药

## 7. 常见误区

### 误区 1：把计分卡当成指标的堆砌

> "有些管理者认为，只要以客户满意度和市场占有率等非财务指标来补充财务指标，平衡计分卡就算大功告成了。但是，他们选择的非财务指标本身也带有不少传统财务指标的缺陷。这些指标是滞后指标，报告的是企业战略过去的实施情况。而且，它们是通用性指标，代表了所有企业都试图改进的方面。"

**纠正**：平衡计分卡不是 20-25 个指标的随意集合，而是通过因果关系链把战略翻译成指标。每个指标都是因果链的一环，最终指向财务目标。

### 误区 2：财务指标过时了，可以不看

> "并非每个长期战略都是获利的。数字设备公司和通用汽车公司在 20 世纪 80 年代并不缺乏长期战略，它们在先进的制造技术、质量和研究与开发上曾投入巨资。但是，它们的愿景和追求成功的经营模式并不符合当时的市场要求。当经营业绩的提高不能使财务业绩有所提高时，高层管理者就必须重新考虑公司的战略或实施计划。"

**纠正**：平衡计分卡的所有因果关系链必须以财务目标为终点。质量改进、客户满意度如果不能转化为财务结果，就不是真正的成功。财务指标不是敌人，它是终极检验标准。

### 误区 3：对所有人都用同一套财务指标

> "许多企业习惯于对所有部门和业务单位采用相同的财务目标。虽然这种统一的做法肯定是可行的、整齐划一的，从某种意义上讲还是'公平的'，因为它用同样的指标来评价每个业务单位管理层的表现。但是它没有考虑不同的业务单位可能所遵循的战略不同。"

**纠正**：成长期、保持期、成熟期的业务单位，财务目标应该完全不同。用同一个 ROI 考核所有业务单位，等于让正在拓荒的团队和正在收割的团队比利润率。

### 误区 4：把计分卡开发委派给中层经理

> "我们最怕接到这样的电话：'喂，我是某某公司的助理主计长，目前在公司负责领导一个改进业绩衡量系统的工作小组。我们想了解一下其他公司用得最成功的是哪一类指标。'计分卡开发流程不应该委派给中层经理组成的工作小组。要想使平衡计分卡发挥效力，它必须反映高级管理层的战略愿景。"

**纠正**：平衡计分卡的最高负责人必须是 CEO 和高级管理层。委派给中层的结果是：做出来的是一套 KPI 集合，不是战略翻译，而且高层根本不会用它来管理。

### 误区 5：追求完美才迟迟不行动

> "每当我们发现企业因为不敢肯定是否选对指标，或因为某些指标的数据不全而踯躅不前时，我们的建议是：'干吧。'先开展学习流程，学会如何以一套平衡的业绩驱动因素和结果指标来管理公司。"

**纠正**：平衡计分卡不是永恒不变的，它是动态的，应该持续研讨、评价和更新。用"目前最好的信息"起步，边用边改，比等完美指标等一年要强得多。

### 误区 6：用拙劣手段完成指标

> "如果对单一的客户指标'按时交货'施加太大压力，管理者很快会开发出机能失调的手段来实现优秀的按时交货指标。例如，制造业者可能大量存储所有可能需要的产品部件，因此所有需求都可以从库存产成品中得到满足。这些企业可能有极佳的按时交货指标，但是大量资本被运用在存货、仓储和搬运设备上。"

**纠正**：每个战略指标都可能被"拙劣地完成"。按时交货可以用大量存货实现，新产品上市时间可以用微小改进推出。必须用互补指标平衡——比如用存货周转率平衡按时交货，用新产品毛利率平衡上市速度。

### 误区 7：只用通用滞后指标冒充平衡计分卡

> "许多高层管理者认为，只要以客户满意度和市场占有率等非财务指标来补充财务指标，平衡计分卡就算大功告成了。但是，他们选择的非财务指标本身也带有不少传统财务指标的缺陷。这些指标是滞后指标，报告的是企业战略过去的实施情况。而且，它们是通用性指标，代表了所有企业都试图改进的方面。这些指标用来计分还是不错的，但是用来告诉员工如何在未来的竞争中立于不败之地，却显得力不从心。"

**纠正**：这是书中第十二章明确总结的"结构缺陷"。一个真正的平衡计分卡必须包含**成果指标和业绩驱动因素的混合**、**滞后指标和领先指标的混合**，而且所有指标都应源自**特定的战略**，而非通用的最佳实践。只用市场份额、客户满意度等通用指标，等于只装了仪表盘上的转速表，没装导航仪。

## 8. 需要谨慎处理的讲师观点

| 观点 | 类型 | 说明 |
|------|------|------|
| "平衡计分卡是信息时代企业的管理基础" | **核心主张** | 卡普兰和诺顿的核心信念：工业时代的管理系统（财务控制）已不适应信息时代。这是全书的底层哲学，不是可验证的事实 |
| "四个层面足够了吗？我们尚未见到任何公司采用的平衡计分卡少于四个层面" | **经验主张** | 基于他们的咨询实践，但明确承认"没有任何数学定理可以证明这四个层面既是必要的又是充分的"——这是谨慎的，不是教条的 |
| 战略业务单位（SBU）是实施计分卡的理想组织单位 | **方法论框架** | 这是他们设计的方法论前提，但书中也承认公司级、职能部门、非营利组织都可以用。选择 SBU 是"最容易成功"而非"唯一正确" |
| "平衡计分卡主要是战略实施的机制，而不是战略制定的机制" | **核心主张** | 这是全书最关键的边界声明。如果战略本身是错的，计分卡只会让组织更快地走向错误的方向。但书中对此着墨不多 |
| 双循环学习是战略成功的核心 | **核心主张** | 他们认为这是计分卡"最具创新性也最为重要的方面"。但 1996 年的技术条件（群组软件、电子公告栏）严重限制了双循环学习的落地，这个主张在当时更多是愿景而非现实 |
| 服务利润链：员工满意度→客户满意度→财务业绩 | **引用观点** | 引用自 Heskett 等人的研究，不是卡普兰和诺顿的原创发现。但他们在书中多处用这个链条来验证计分卡因果关系的合理性 |
| 奖金与计分卡挂钩应谨慎 | **经验主张** | 他们观察到急于挂钩的风险，建议先用门槛条件而非简单加权。但同时也承认"至今仍无定论"，这是诚实的经验总结而非确定结论 |
| 战略制定与实施可以自下而上涌现 | **引用观点** | 引用自明茨伯格和西蒙的"新兴战略观"，承认战略可能来自基层。这与他们"自上而下"的计分卡构建流程存在一定张力 |

### 书中五个贯穿案例

这五家公司是全书的教学骨架，每章都在用它们示范不同层面的构建和应用：

| 公司 | 行业 | 实施驱动力 | 关键做法 |
|------|------|-----------|---------|
| **石水公司** | 海底建筑工程 | 两家公司合并后统一文化和战略：从低价竞争转向增值客户关系 | 区分一级客户（增值合作）和二级客户（价格导向）；开发了 16 项客户满意度加权评分；通过改进项目管理流程解决应收账款过长问题 |
| **大都会银行** | 金融服务（银行） | 两家银行合并后统一战略：从交易导向转向目标客户理财顾问 | 从 70+ 个行动方案砍到少数战略方案；开发交叉销售率、面对客户时间等领先指标；用作业成本法衡量客户获利率 |
| **先锋石油公司** | 石油营销与炼油 | 分散集中化的职能权力，培养 14 个新 SBU 领导者的战略能力 | 60% 奖金与财务指标挂钩，40% 与非财务指标挂钩；区分经销商（中间客户）和最终消费者两类客户；用"神秘客"评分衡量消费者体验 |
| **国家保险公司** | 财产与意外保险 | 扭亏为盈：从全市场全服务转型为专业专项保险 | 30 个月完整实施；承保质量审计得分作为领先指标（理赔结果滞后数年）；从总公司到 SBU 到个人层层下达 |
| **凯尼恩商店** | 服装零售 | 从高度自治向总公司战略协同转型：实现成长和利润率提升 | 五个关键内部流程（品牌管理、时尚领导、货源领导、商品供应、购物体验）；用情景规划法将"收入五年翻倍"的目标分解为可操作的驱动因素；制定全面沟通计划（不同对象不同渠道） |
